## ✅ Build Order — What to Write First

```
ROUND 1 — Core Coding Standards (reusable forever, write once)

6.  instructions/pages.md                  ← BA & PM workflow — image → requirements → tasks

ROUND 2 — Conditional Instruction Files (write once, reused when triggered)
────────────────────────────────────────────────────────────────────────────
7.  instructions/async.md                  ← @Async, @Scheduled, job status tracking

11. instructions/notification.md           ← FCM / VAPID — checks stack.md first
12. instructions/whatsapp.md               ← WA Cloud API / Twilio — checks stack.md first
13. instructions/websocket.md              ← STOMP / SSE — checks stack.md first

ROUND 3 — Agent Execution Commands (OpenClaw workflow)
──────────────────────────────────────────────────────
14. commands/AI-AGENT-EXECUTION-GUIDE.md   ← Most important file — OpenClaw reads first
15. commands/execute-task.md
16. commands/continue-tasks.md

18. commands/fix-task.md

ROUND 4 — Trackers
────────────────────
24. .claude/configurations.md              ← Config tracker (✅ READY / ❌ MISSING)

    → commands/speckit.constitution.md
    → commands/speckit.specify.md

    → commands/speckit.tasks.md
    → commands/speckit.implement.md

The agent reads these files automatically based on signals in the spec.

```

## 🚀 New Project Setup — Exact Steps

```bash
# Step 1 — Copy template to new project
cp -r claude-project-template/.claude ./new-project/.claude
cp -r claude-project-template/designs ./new-project/designs

# Step 2 — Trackers are pre-configured — nothing to reset
# .claude/configurations.md → all configs start as ❌ MISSING (ready to use)
# .claude/tasks.md          → empty — agent fills as pages are analyzed

# Step 3 — Fill 3 project files (10–30 minutes total)
# Edit: .claude/project/stack.md    → your tech stack + providers
# Edit: .claude/project/users.md    → your user types + permissions
# Edit: .claude/project/pages.md    → your page list + access + API mapping

# Step 4 — Write CLAUDE.md for this project
# Edit: .claude/CLAUDE.md           → project name, modules, phases, budget

# Step 5 — Add design images + page specs
# For each page:
#   Copy designs/PAGE_TEMPLATE.md → designs/1.md (or any number)
#   Fill in: page name, business, access, permissions, notes
#   Drop design image: designs/1.png (same number as .md file)
#   Tell agent: "Analyze designs/1.png" → requirements + tasks auto-generated

# Step 6 — Run SpecKit
specify init . --ai claude
# Then in Claude Code:
# /speckit.constitution  → set project principles
# /speckit.specify       → generate spec per module
# /speckit.plan          → generate plan per module
# /speckit.tasks         → generate tasks → feeds systemTasks.md

# Step 7 — Run OpenClaw
# /execute-task          → agent reads everything and starts building
# /continue-tasks        → keep going until all tasks done
```
