### How Phases Are Generated

After the agent analyzes all design images, it groups tasks into phases:

```

  → Project scaffold, DB setup, Docker Compose
  → Security, JWT auth, CORS config
  → Base entity, ApiResponse, GlobalExceptionHandler

Phase 2 to N — One Phase Per Module
  → Backend task before Frontend task in same module
  → Modules ordered by dependency
    (auth before everything, upload before image-heavy content)

Final Phase — Integration
  → Navigation wiring across all pages
  → Regression tests
  → Performance and SEO review
```

The agent fills in the table above and updates the Current Status section

## Architecture Rules

### Backend
1. All `/api/admin/**` endpoints require JWT — `@PreAuthorize` on every method
2. All `/api/public/**` endpoints are open — no auth required
3. DTOs only in controllers — entities never returned from controllers
4. `@Transactional` on service layer only — never on controllers
5. Flyway migrations only — `spring.jpa.hibernate.ddl-auto=validate`
6. Redis cache invalidated on every write operation
7. Soft delete only — `entity.setDeletedAt(Instant.now())` — never hard delete
8. Slug auto-generated from title in service layer
9. Reading time auto-calculated from word count in service layer
10. Rate limiting on all public POST endpoints via Redis

### Frontend
11. RTK Query for all API calls — no raw fetch or axios outside RTK Query
12. No hardcoded API URLs — `import.meta.env.VITE_API_BASE_URL` only
13. All admin routes wrapped in `AuthGuard` component
14. Handle loading, error, and empty states on every API-connected component
15. Logical CSS properties — `ms-*` not `ml-*`, `text-start` not `text-left`
16. Mobile first — default breakpoint is mobile, scale up
17. All `data-testid` attributes on every interactive element

## API Response Format

Every backend response uses this format:

**Success:**
```json
{
  "success": true,
  "data": { },
  "timestamp": "2026-03-09T10:00:00Z"
}
```

**Error:**
```json
{
  "success": false,
  "error": {
    "code": "RESOURCE_NOT_FOUND",
    "message": "Blog not found"
  },
  "timestamp": "2026-03-09T10:00:00Z"
}
```

## Security

```
Public routes:  /api/public/**  → no auth
Auth routes:    /api/auth/**    → no auth (login endpoint)
Admin routes:   /api/admin/**   → JWT required

Roles:
  ROLE_ADMIN   → full access to all admin endpoints
  ROLE_EDITOR  → blog and case study management only

JWT secret:     environment variable JWT_SECRET
Token expiry:   1 hour (access) + 7 days (refresh)
Blacklist:      Redis (on logout)
```
